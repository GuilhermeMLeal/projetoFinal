Tipos de redes:

    PAN - Personal Area Network(Rede de Área Pessoal) = Essa rede tem a função ligar aparelhos de uso pessoal,através de uma conexão sem fio.

    CAN - Controller Area Network (Rede de Área de Controladores) = Tipo de rede de comunicação usado principalmente em sistemas automotivos.

    LAN - Local Area Network (Rede de Área Local) = Rede de computadores que abrange uma área geograficamente limitada, como uma casa, escritório ou edifício.

    MAN - Metropolitan Area Network (Rede de Área Metropolitana) = Conecta várias LANs dentro de uma cidade ou região metropolitana

    WAN - Wide Area Network (Rede de Área Ampla) = Estende por uma grande área geográfica, como um país, continente ou até mesmo globalmente(engloba LANs e MANs ).

Topologias de redes:

Topologia é um termo utilizado para designar a configuração assumida pelos computadores de uma Rede, ou seja, a forma como estes estão interligados.

    Topologia Física = Refere-se à forma como os computadores e cabos de rede se interligam fisicamente(Estrela,Barramento e Anel) na Rede. Topologia Lógica = É a troca de dados entre dispositivos, independentemente da sua conexão física.
  • Topologia de Barramento = Todos os dispositivos são conectados a um único cabo em linha reta, chamado de barramento.


  • Topologia em Anel = Os dispositivos são conectados em um formato de anel, onde cada dispositivo está conectado a outros dois dispositivos e ele pode percorrer por um único sentido.


  • Topologia em Estrela = Todos os dispositivos são conectados a um dispositivo central(switch) e a partir dele vai enviar a informação para o dispositivo requisitado e devolver para quer fez a requisição .


    • Topologia em Estrela Estendida: É uma variação da topologia em estrela usada para conectar várias redes locais (LANs) em uma rede maior.

  • Topologia em Malha = Cada dispositivo está conectado a todos os outros dispositivos da rede, ou seja, você não precisa de um "intermediário".

Modelo OSI e Tanenbaum

O Modelo OSI em si

Camada 1 - Física: Camada física é o caminho que os pacotes percorrem para chegar ao destino(transmissão fibra óptica, wireless e cabos de rede convencionais(cobre)), ou seja, os dados são transmitidos e processados , adiante são passados para a próxima camada .

Camada 2 - Enlace ou Ligação (Fiscal): Ele observa se o pacote(meio físico) tem algum defeito em sua formatação e controla o fluxo com que os pacotes transmitidos para camadas superiores, após essa análise os dados recebidos assumem uma transmissão praticamente sem erros. Essa camada é conhecida pela função dos switches e é dividida em 2 subcamadas:

    MAC(Media Acces Control / Controle de acesso a midia): Camada que possibilita a conexão de diversos computadores em uma rede. Cada máquina conectada na rede tem um endereço físico, conhecido como endereço MAC e por meio desse endereço que se identifica e envia os pacotes. LLC(Logical Link Control / Controle de link lógico) : Camada que temos o controle de fluxo dos dados na rede para a próxima camada. FCS (Frame Check Sequence / Sequência de Verificação de Quadro)

Camada 3 - Rede : Verifica o emissor e receptor da mensagem. Se existirem muitas requisições para serem enviadas, eles podem priorizar quais serão enviadas primeiro e qual é o melhor caminho para enviar esses dados. Basicamente, ela controla o roteamento entre a origem e destino do pacote e após definição vai para o transporte.

Diferença entre IP e MAC:

    MAC é o endereço físico de quem envia o pacote, ou seja, no envio nós passamos por vários dispositivos(roteador, switch, servidor e entre outros) e cada um tem seu MAC, ou seja, o endereço vai se mudar constantemente e até se perdendo no caminho de retorno. IP é um indereço no qual não sofre alteração

Máscara de Rede :

Camada 4 - Transporte: Camada que garante o envio e o recebimento dos pacotes vindos da camada 3. Ela gerencia o transporte dos pacotes para garantir o sucesso no envio e no recebimento de dados, com preocupação para que os dados sejam entregues com consistência, sem erros e duplicações.
