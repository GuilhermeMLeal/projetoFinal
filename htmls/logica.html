HTML E CSS

O HTML5(Hypertext Markup Language / linguagem de marcação) é aquele conhecido como linguagem de marcação e basicamente ele representa um esquelo da aplicação Web. A escrita é feita por tags, nas quais podemos fazer uma escrita, tabela, campos de entrada de informações fornecidas por usuários e muitos outros.

O CSS3(Cascading Style Sheets / linguagem de design gráfico) representa basicamente a estilização do HTML, no qual este é utilizado para gerar páginas mais atrativas e com interatividade. Assim como seu nome, esse processo é feito de modo cascata, ou seja efetua a leitura da codificação de cima para a última linha.

HTML Semântico

Diferentemente do HTML normal, utilizado no exercício do Hamburguer, no HTML semântico nós damos significância a estrutura da página. Mas como assim? Basicamente, a página é dividida em diferentes estruturas, sendo elas as mais comuns: header (cabeçalho), nav(barra de navegação, normalmente acompanha o header e aside), main(meio e informações gerais), aside(barra de complemento) e footer(roda pé).

Outro complemento em um site organizado e estruturado, seria a utilização de centralizações e interatividades no CSS, através do display:flex, flex-direction, justify-content, align-itens ou align-content. A forma mais comleta e útil, também seria a utilização de margin(margem), padding(espaçamento interno), cores e modificações de fundo.

Operadores e Operações em JavaScript

Antes dos operadores devemos relembrar das operações, sendo elas:

IF, ELSE, ELSE IF, TERNÁRIO E SWITCH CASE

Basicamente, if(se) é usado para executar um bloco de código se uma condição especificada for verdadeira, else if (senão se) é usado quando você tem várias condições e deseja verificar cada uma delas sequencialmente, else (senão) é usado quando nenhuma das condições anteriores é verdadeira. O operador ternário é a mesma representação do if e else, pois é como se fosse feita uma pergunta, na qual a reposta verdadeira é antes dos ":" e a falsa é depois dos ":"

Switch Case não foge muito do if e else, porém ele serve como uma alternativa para mostrar várias condições em paralelo e verificar a melhor condição viável.

While, Do While e For

As estruturas de repetição mais comuns são o "for", "while" e "do-while". O for é usado quando você sabe exatamente quantas vezes deseja repetir um bloco de código, ele é representado como: for = {inicialização,condição e atualização}, while é usado quando você deseja repetir um bloco de código enquanto uma condição especificada for verdadeira e do-while é semelhante ao "while", mas a condição é verificada após a execução do bloco de código.
