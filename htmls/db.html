Banco de Dados

Afinal, o que são?

Banco de Dados ou Data Base são estruturas físicas, nas quais dados e informações são mantidas por um período incerto de tempo e são utilizadas para algum propósito, principalmente feito para lidar com grandes volumes de informações e oferecer recursos para garantir a integridade, segurança e disponibilidade dos dados de um algumas instituição.

MER e DER

MER é um modelo conceitual utilizado para descrever os objetos (entidades) envolvidos em um domínio de negócios, com suas características (atributos) e como elas se relacionam entre si (relacionamentos).

Entidade

Entidades:Os objetos ou partes envolvidas um domínio, também chamados de entidades, podem ser classificados como físicos ou lógicos, de acordo sua existência no mundo real.

Relacionamento

Relacionamento é basicamente a interação de entidades, são nomeados com verbos ou expressões que representam a ação de uma para outra, como também seu relacionamento.Esse relacionamento pode ser representado de duas formas: Forte e Fraco

    Relacionamento forte = Se a entidade a qual possui um relacionamento for uma entidade com valor igual(forte - forte / fraco - fraco) usa-se um RELACIONAMENTO FORTE Relacionamento fraco = Caso se a entidade for distinta (forte - fraca)

Além, da Cardinalidade que é a relação de entidade para entidade, juntamenet com o relacionamento.

Atributo

É uma característica ou propriedade que descreve uma entidade ou objeto. Em termos mais simples, um atributo é uma coluna em uma tabela de banco de dados. Cada registro (linha) em uma tabela terá valores correspondentes para cada atributo.

DQL, DDL e DML

DDL (Data Definition Language): É uma linguagem usada para definir a estrutura e a organização dos dados em um banco de dados. As principais operações do DDL são:

    CREATE: Usado para criar tabelas, índices, visões e outros objetos do banco de dados. ALTER: Utilizado para modificar a estrutura de objetos existentes no banco de dados, como adicionar ou remover colunas de uma tabela. DROP: Usado para excluir objetos do banco de dados, como tabelas ou índices.

DML (Data Manipulation Language): É uma linguagem usada para manipular os dados em um banco de dados. As principais operações do DML são:

    INSERT: Usado para inserir novos registros em uma tabela. UPDATE: Usado para atualizar os valores existentes em uma tabela. DELETE: Usado para excluir registros de uma tabela.

DQL (Data Query Language): É uma linguagem usada para consultar dados em um banco de dados. A principal operação do DQL é:

    SELECT: Usado para recuperar dados de uma ou mais tabelas do banco de dados, especificando as colunas desejadas, condições de filtro e opções de junção.

3 formas normais para um Banco de Dados

1º forma normal

Não posso ter campos multivalorados, nem compostos (Atributos)

2º forma normal

Não posso conter dependência parcial( atributos que não são chave dependem unicamente da chave primária da tabela ) e deve pertencer a 1º forma normal.

3º forma normal

Não contenho dependência transitiva
